1008 (2018) - Uniform Underwriting and Transmittal Summary
Suggest Edits

This is a recent version of the standardized form used in the mortgage industry to facilitate the underwriting process of a loan application. This document compiles essential information about the borrower, the requested loan, the property involved, and the underwriter's decision-making basis.

To use the Upload PDF endpoint for this document, you must use 1008_2018 in the form_type parameter. To learn more about processing this document, click here.
