4868 - Application for Automatic Extension of Time to File U.S. Individual Income Tax Return
Suggest Edits

U.S. taxpayers use this form to request an automatic six-month extension for filing their individual income tax returns. It extends the filing deadline but does not extend the time to pay any taxes owed.

To use the Upload PDF endpoint for this document, you must use A_4868 in the form_type parameter.
